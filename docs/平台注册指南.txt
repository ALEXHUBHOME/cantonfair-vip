# 社交媒体平台注册指南

## 一、国内平台

### 1. 小红书

**注册方式：** - 手机APP注册（推荐） - 需要中国大陆手机号 - 实名认证可选

**企业号注册：** 1. 下载小红书APP 2.
点击"我"-"设置"-"账号与安全"-"申请认证" 3. 选择"企业号" 4. 提交营业执照
5. 审核时间：3-5工作日

**建议：** 先用个人号测试内容，企业号适合有稳定内容产出后

------------------------------------------------------------------------

### 2. 抖音

**注册方式：** - 手机号注册 - 可升级企业号

**企业号权益：** - 蓝V认证 - 数据分析 - 官网链接

**入驻流程：** 1. 下载抖音APP 2. 手机号注册 3. 我-设置-申请官方认证 4.
企业认证-提交资料 5. 费用：600元/年

------------------------------------------------------------------------

### 3. 微信公众号

**注册方式：** 1. 访问微信公众平台 https://mp.weixin.qq.com 2.
选择注册类型： - 订阅号（适合内容） - 服务号（适合业务） 3. 邮箱激活 4.
填写资料 5. 微信扫码验证

**企业号（企业微信）：** - 需营业执照 - 可添加多个员工 - 适合客服

------------------------------------------------------------------------

## 二、国际平台

### 1. TikTok

**个人号注册：** 1. 下载TikTok APP 2. 选择注册方式： - 邮箱注册 -
Google账号 - Facebook账号 3. 填写生日（需18+） 4. 验证邮箱

**企业号：** 1. 切换到Business账号 2. 设置商业信息 3. 免费！

------------------------------------------------------------------------

### 2. Instagram

**注册方式：** 1. 下载Instagram APP 2. 注册方式： - 邮箱注册 -
Facebook账号关联 3. 设置用户名和头像

**企业账号：** 1. 已绑定Facebook主页 2. 设置-账号-切换为商业账号 3.
完善商业信息 4. 免费！

------------------------------------------------------------------------

### 3. Facebook

**注册方式：** 1. 访问 facebook.com 2. 创建账户 3. 填写基本信息 4.
验证邮箱/手机

**公共主页（类似企业号）：** 1. 创建-公共主页 2. 填写名称和类别 3.
添加头像和封面 4. 完善信息 5. 免费！

------------------------------------------------------------------------

### 4. YouTube

**注册方式：** 1. 访问 youtube.com 2. 点击"Sign In" 3. 创建频道 4.
填写频道名称

**盈利条件：** - 1000订阅者 - 4000观看小时 - 即可申请盈利

------------------------------------------------------------------------

### 5. Twitter/X

**注册方式：** 1. 访问 twitter.com 2. 创建账户 3. 邮箱或手机验证

**企业认证：** - 需要蓝勾（\$8/月） - 或申请企业认证（需符合条件）

------------------------------------------------------------------------

## 三、账号运营建议

### 优先顺序

1.  **小红书** - 精准触达中文用户
2.  **TikTok** - 国际化第一步
3.  **抖音** - 国内短视频流量
4.  **YouTube** - 长视频和SEO
5.  **Instagram** - 国际化图片

### 内容策略

-   小红书：图文+短视频
-   TikTok：英语短视频
-   YouTube：长视频教程
-   Instagram：高质量图片

------------------------------------------------------------------------

## 四、各平台禁止内容（需注意）

  平台       禁止内容
  ---------- --------------------
  小红书     敏感词、违规广告
  抖音       过度营销、敏感内容
  TikTok     政治敏感、版权内容
  Facebook   仇恨言论、虚假信息
  YouTube    版权视频、违规内容
